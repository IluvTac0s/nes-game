| Instruction             | Description             |
| ----------------------- | ----------------------- |
| `LDX #imm`              | Load X                  |
| `LDY #imm`              | Load Y                  |
| `STX addr`              | Store X                 |
| `STY addr`              | Store Y                 |
| `INX` / `DEX`           | Increment / decrement X |
| `INY` / `DEY`           | Increment / decrement Y |
| `CPX #imm` / `CPY #imm` | Compare X/Y with value  |
