| Instruction   | Description                               |
| ------------- | ----------------------------------------- |
| `NOP`         | No operation                              |
| `BRK`         | Force interrupt                           |
| `CLC` / `SEC` | Clear / Set Carry                         |
| `CLD` / `SED` | Clear / Set Decimal mode (ignored in NES) |
| `CLI` / `SEI` | Clear / Set Interrupt disable             |
